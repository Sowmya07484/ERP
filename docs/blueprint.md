# **App Name**: Quantis ERP

## Core Features:

- Live KPI Command Center: An interactive dashboard visualizing real-time metrics including revenue, profit, stock value, and active production status with dynamic chart overlays.
- Lifecycle Order Management: Complete workflow automation from quotation to invoice, featuring automatic calculations for GST, HSN, and multi-tier discounts.
- Automated BOM & Production Engine: Calculates bill of materials costs and tracks production stages from planned to completed while updating inventory values automatically.
- Smart Procurement & Reorder Tracker: Automated low-stock alerts based on threshold data that generate purchase requests and track material delivery status in a unified timeline view.
- Double-Entry Financial Ledger: A robust accounting module that automates journal entries for every transaction, feeding directly into Trial Balance and Balance Sheet reports.
- Intelligent Financial Forecast Tool: A generative AI-powered tool that analyzes ledger history and inventory levels to predict cash flow and recommend procurement schedules.
- Role-Based Audit Console: Secure access control for 8 unique roles with full activity logging and historical audit trails for sensitive financial and data operations.

## Style Guidelines:

- Primary color: Vibrant Safety Orange (#F97316) used for high-importance interactions and primary CTA buttons.
- Background color: Soft Slate Gray (#F1F5F9) to provide a neutral, clean, and professional workspace environment.
- Accent color: Deep Burnt Sienna (#EA580C) used for interactive states like hovers to maintain an analogous orange-based palette.
- Font pairing: 'Space Grotesk' for data-heavy headlines and KPIs for a machined look; 'Inter' for readable body text and high-density tables.
- Professional, geometric stroke icons using thin line-weights to denote industrial sectors (gears, pallets, dollar signs).
- Enterprise-grid layout with a collapsible sidebar and card-based data modules designed for a data-dense single-page application feel.
- Subtle slide-and-fade transitions when switching between ERP modules to ensure the interface feels snappy and efficient.
